# ADR-0001 - Fundação do Projeto

## Status

Aceito

## Contexto

O YouMail será uma plataforma SaaS de hospedagem de e-mails multiempresa.

## Decisão

Separar a infraestrutura do servidor de e-mail da API de gerenciamento.

## Consequências

- Infraestrutura independente
- API independente
- Maior escalabilidade
- Melhor manutenção
