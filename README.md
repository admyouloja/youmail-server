# YouMail Server

Servidor de e-mail SaaS desenvolvido pela YouSaaS.

## Objetivo

Fornecer uma plataforma completa de hospedagem de e-mails para múltiplos domínios, permitindo que empresas utilizem seus próprios domínios com caixas postais profissionais.

### Exemplos

- contato@empresa.com.br
- financeiro@empresa.com.br
- suporte@empresa.com.br

## Tecnologias previstas

- Docker
- Traefik
- Postfix
- Dovecot
- MariaDB
- Roundcube
- Rspamd
- Redis
- API própria
- Base44
- Supabase
